## About ##

Simple Pagination component for IndexedDB.

## Technologies ##

The following tools were used in this project:

- [idbx](https://github.com/xpectme/idbx) - an indexedDB wrapper

## Getting started ##

```ts
import * as idbx from "https://deno.land/x/idbx/main.ts";
```
